<summary><b>Source Control Graph colors(ソースコントロールグラフカラー)</b></summary><br>


**Source Control Graph colors** は、サイドバーのソース管理ビュー(`Ctrl`+`Shift`+`G`)に表示されるコミット履歴グラフ(GRAPH)に関する色設定である。
